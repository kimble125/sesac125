## 🚀 실행 방법

### React 프로젝트 실행

```bash
# 프로젝트 디렉토리로 이동
cd 05-frontend-development/myReact/react-project-01

# 의존성 설치
npm install

# 개발 서버 실행
npm run dev
```

### JavaScript 기초 실습 파일 실행

```bash
# 기초 문법 예제 실행
node 05-frontend-development/01_기초문법.js
```
